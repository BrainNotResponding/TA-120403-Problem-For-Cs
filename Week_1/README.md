# 📂 แบบฝึกหัดการวิเคราะห์ปัญหาทางคอมพิวเตอร์ (Input - Process - Output)

คลังเก็บงานนี้รวบรวมการวิเคราะห์โจทย์ปัญหาพื้นฐานทางวิทยาการคอมพิวเตอร์ โดยใช้วิธีการแยกแยะองค์ประกอบแบบ **Input (ข้อมูลเข้า)**, **Process (การประมวลผล)** และ **Output (ข้อมูลออก)** เพื่อเตรียมความพร้อมก่อนการนำไปพัฒนาเป็นโปรแกรมจริง

---

## 📋 ตารางวิเคราะห์โจทย์ปัญหา (IPO Table)

### • 1. โปรแกรมคำนวณปริมาตรกล่องทรงสี่เหลี่ยมมุมฉาก

| Input (ข้อมูลเข้า) | Process (การประมวลผล) | Output (ผลลัพธ์) |
| :--- | :--- | :--- |
| • ค่าความกว้างของกล่องทรงสี่เหลี่ยมมุมฉาก<br>• ค่าความยาวของกล่องทรงสี่เหลี่ยมมุมฉาก<br>• ค่าความสูงของกล่องทรงสี่เหลี่ยมมุมฉาก | • **อ่าน:**<br>&nbsp;&nbsp;- ค่าความกว้างของกล่อง<br>&nbsp;&nbsp;- ค่าความยาวของกล่อง<br>&nbsp;&nbsp;- ค่าความสูงของกล่อง<br>• **คำนวณ:สูตร`กว้าง × ยาว × สูง`**<br>&nbsp;&nbsp;`ค่าความกว้างของกล่องทรงสี่เหลี่ยมมุมฉาก ×  ค่าความยาวของกล่องทรงสี่เหลี่ยมมุมฉาก × ค่าความสูงของกล่องทรงสี่เหลี่ยมมุมฉาก`<br>• **พิมพ์:**<br>&nbsp;&nbsp;- ค่าปริมาตรของกล่องทรงสี่เหลี่ยมมุมฉาก | • ค่าปริมาตรของกล่องทรงสี่เหลี่ยมมุมฉาก |

## 🧠 ความรู้เพิ่มเติมที่ได้รับ
* **Input-Process-Output (IPO Chain):** เป็นพื้นฐานสำคัญในการออกแบบอัลกอริทึม ช่วยให้ผู้เขียนโปรแกรมไม่สับสนว่าต้องรับค่าอะไร และต้องแสดงผลลัพธ์แบบไหน
* **เครื่องมือในการทดสอบ:** 
  * **Compiler:** ตัวแปลโปรแกรมที่แปลโค้ดทั้งหมดในคราวเดียว (เช่น C++)
  * **Interpreter:** ตัวแปลโปรแกรมที่แปลและทำงานทีละบรรทัด (เช่น Python)
