# 📂 แบบฝึกหัดการวิเคราะห์ปัญหาทางคอมพิวเตอร์ (Input - Process - Output)

คลังเก็บงานนี้รวบรวมการวิเคราะห์โจทย์ปัญหาพื้นฐานทางวิทยาการคอมพิวเตอร์ โดยใช้วิธีการแยกแยะองค์ประกอบแบบ **Input (ข้อมูลเข้า)**, **Process (การประมวลผล)** และ **Output (ข้อมูลออก)** เพื่อเตรียมความพร้อมก่อนการนำไปพัฒนาเป็นโปรแกรมจริง

---

## 📋 ตารางวิเคราะห์โจทย์ปัญหา (IPO Table)
1. โปรแกรมคำนวณปริมาตรกล่องทรงสี่เหลี่ยมมุมฉาก
| Input (ข้อมูลเข้า) | Process (การประมวลผล) | Output (ผลลัพธ์) |
| :--- | :--- | :--- |
| • ค่าความกว้างของกล่อง<br>• ค่าความยาวของกล่อง<br>• ค่าความสูงของกล่อง | อ่าน <br> • ค่าความกว้างของกล่อง<br> • ค่าความยาวของกล่อง<br>• ค่าความสูงของกล่อง<br>• คำนวณตามสูตร:<br>`กว้าง × ยาว × สูง`<br> พิมพ์<br> • ค่าปริมาตรของกล่องทรงสี่เหลี่ยมมุมฉาก | • ค่าปริมาตรของกล่องทรงสี่เหลี่ยมมุมฉาก |

| **2. โปรแกรมคำนวณพื้นที่สามเหลี่ยม** | • ความสูงของสามเหลี่ยม<br>• ความยาวฐานของสามเหลี่ยม | • คำนวณตามสูตร:<br>`0.5 × ฐาน × สูง` | • พื้นที่สามเหลี่ยม |
| **3. โปรแกรมคำนวณภาษีรายได้** | • รายได้ทั้งหมด | • คำนวณภาษีตามอัตราที่กำหนด | • จำนวนภาษีที่ต้องจ่าย |
| **4. โปรแกรมแปลงเสียงพูดเป็นข้อความ** | • ไฟล์เสียง หรือ สัญญาณเสียงพูด | • ประมวลผลและแปลงสัญญาณเสียงเป็นตัวอักษร | • ข้อความ (Text) |
| **5. โปรแกรมวิเคราะห์และแยกแยะเพศ** | • ข้อมูลหรือภาพถ่ายของผู้ใช้ | • วิเคราะห์เอกลักษณ์เพื่อแยกเพศ | • ผลลัพธ์เพศ (ชาย / หญิง) |
| **6. โปรแกรมจัดสรรเงินโบนัสให้ครอบครัว** | • จำนวนเงินโบนัสทั้งหมด | 1. แบ่งให้แม่ตามสัดส่วนที่กำหนด<br>2. นำเงินที่เหลือมาหารแบ่งให้ลูกแต่ละคนเท่าๆ กัน | • จำนวนเงินที่แม่ได้รับ<br>• จำนวนเงินที่ลูกแต่ละคนได้รับ |
| **7. โปรแกรมคำนวณกำไรและการขายแตงโม** | • จำนวนแตงโมที่ซื้อ<br>• ราคาทุนที่ซื้อมา<br>• ราคาที่ขายไป<br>• จำนวนแตงโมที่เหลือ (เน่าเสีย) | 1. หาจำนวนผลที่ขายได้จริง<br>2. คำนวณเงินที่ได้: `ผลที่ขายได้ × ราคาขาย`<br>3. คำนวณกำไร: `เงินที่ได้ทั้งหมด - ทุนทั้งหมด` | • กำไรทั้งหมดจากการขายแตงโม<br>• จำนวนแตงโมที่เหลือหรือหมดไป |
| **8. โปรแกรมเก็บภาษีและสรุปรายได้ต่อวัน (สถานบริการ)** | • จำนวนลิตร (หรือปริมาณบริการ) ที่ได้ใน 1 วัน | • คำนวณภาษีสะสม<br>• คำนวณรายได้สุทธิหลังหักภาษี | • ยอดภาษีประจำวัน<br>• รายได้คงเหลือสุทธิในหนึ่งวัน |
| **9. โปรแกรมคำนวณพื้นที่สี่เหลี่ยมคางหมู** | • ความสูงของพื้นที่<br>• ความยาวของด้านคู่ขนานเส้นที่ 1<br>• ความยาวของด้านคู่ขนานเส้นที่ 2 | • คำนวณตามสูตร:<br>`0.5 × สูง × (ผลรวมของด้านคู่ขนาน)` | • พื้นที่สี่เหลี่ยมคางหมู |

## 🧠 ความรู้เพิ่มเติมที่ได้รับ
* **Input-Process-Output (IPO Chain):** เป็นพื้นฐานสำคัญในการออกแบบอัลกอริทึม ช่วยให้ผู้เขียนโปรแกรมไม่สับสนว่าต้องรับค่าอะไร และต้องแสดงผลลัพธ์แบบไหน
* **เครื่องมือในการทดสอบ:** 
  * **Compiler:** ตัวแปลโปรแกรมที่แปลโค้ดทั้งหมดในคราวเดียว (เช่น C++)
  * **Interpreter:** ตัวแปลโปรแกรมที่แปลและทำงานทีละบรรทัด (เช่น Python)
