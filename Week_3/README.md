# C++ Basic Input/Output & Data Types Reference

โปรเจกต์นี้จัดทำขึ้นเพื่อรวบรวมและอธิบายการทำงานของคำสั่งรับ-แสดงผลข้อมูลพื้นฐาน (`printf`, `scanf`) รวมถึงการประกาศและใช้งานตัวแปรประเภทต่างๆ ในภาษา C++ (เช่น `int`, `float`, `char`, และ `string`) เพื่อเป็นแนวทางในการเขียนโปรแกรมอย่างถูกต้อง

---

## 📥 1. คำสั่งรับและแสดงผลข้อมูล (Input / Output Functions)

### 🔹 printf()
*   **ทำงานยังไง:** ทำหน้าที่แปลงข้อมูลจากหน่วยความจำให้ออกมาเป็นตัวอักษรแล้วนำไปแสดงผลบนหน้าจอคอนโซล โดยใช้ **Format Specifier** (เช่น `%d`, `%f`) เป็นตัวระบุตำแหน่งและประเภทของข้อมูลที่จะนำมาหยอดลงในข้อความ
*   **ต้องใส่อะไร:** 
    *   ข้อความที่ต้องการแสดงผล (อยู่ภายในเครื่องหมาย `" "`)
    *   Format Specifier (ถ้ามีตัวแปรมาแสดงผลด้วย)
    *   ชื่อตัวแปรที่ต้องการนำค่ามาแสดง (เรียงลำดับจากซ้ายไปขวาตาม Format Specifier)
    *   *ตัวอย่าง:* `printf("Age: %d", age);`
*   **ทำเพื่ออะไร:** เพื่อสื่อสารกับผู้ใช้ แสดงผลลัพธ์การคำนวณ หรือใช้ตรวจสอบค่าของตัวแปรในระหว่างการพัฒนาโปรแกรม (Debugging)

### 🔹 scanf()
*   **ทำงานยังไง:** ทำหน้าที่หยุดรอรับการกดแป้นพิมพ์จากผู้ใช้ จากนั้นจะอ่านค่าเหล่านั้นและนำไปเก็บไว้ในที่อยู่หน่วยความจำ (Memory Address) ของตัวแปรที่กำหนดไว้
*   **ต้องใส่อะไร:** 
    *   Format Specifier เพื่อบอกว่าข้อมูลที่กำลังจะพิมพ์เข้ามาคือประเภทอะไร (อยู่ในเครื่องหมาย `" "`)
    *   เครื่องหมาย `&` (Ampersand) นำหน้าชื่อตัวแปร เพื่อส่งตำแหน่งที่อยู่ของตัวแปรนั้นให้คำนวณถูกจุด (ยกเว้นตัวแปรประเภท Array หรือ String บางชนิดที่ไม่ต้องใช้ `&`)
    *   *ตัวอย่าง:* `scanf("%d", &age);`
*   **ทำเพื่ออะไร:** เพื่อทำให้โปรแกรมมีความยืดหยุ่น สามารถรับข้อมูลใหม่ๆ จากผู้ใช้เข้ามาประมวลผลได้ โดยไม่ต้องล็อกค่าตายตัวไว้ในโค้ด

## 📊 2. การจัดการตัวแปร (Data Types & Variables)

ตารางสรุปคำสั่งตัวแปร, Format Specifier ที่ต้องใช้คู่กับ `printf`/`scanf`, และวัตถุประสงค์การใช้งาน:

| ตัวแปร (Data Type) | Format Specifier | สิ่งที่ต้องใส่ตอนประกาศ | ทำเพื่ออะไร / เหมาะกับข้อมูลแบบไหน |
| :--- | :---: | :--- | :--- |
| **`int`** (Integer) | `%d`  | ตัวเลขจำนวนเต็ม ไม่มีทศนิยม (เช่น `10`, `-5`, `0`) | เก็บข้อมูลที่เป็นจำนวนนับ เช่น อายุ, จำนวนชิ้น, ไอดี |
| **`float`** (Floating-point) | `%f` | ตัวเลขที่มีจุดทศนิยม (เช่น `3.14`, `99.50`) | เก็บข้อมูลที่ต้องการความละเอียด เช่น ราคาสินค้า, น้ำหนัก, ส่วนสูง |
| **`char`** (Character) | `%c` | ตัวอักษรเดี่ยว 1 ตัว ภายในเครื่องหมาย `' '` (เช่น `'A'`, `'9'`, `'+'`) | เก็บอักขระเดี่ยว รหัสผ่านสัญลักษณ์ หรือเกรดวิชาเรียน |
| **`string`** / **`char[]`** (String) | `%s` | ข้อความยาวๆ ภายในเครื่องหมาย `" "` (เช่น `"Hello World"`) | เก็บข้อมูลที่เป็นคำ ข้อความ หรือประโยค เช่น ชื่อ-นามสกุล, ที่อยู่ |

## 💻 3. เจาะลึกความแตกต่างของ char และ String

ใน C++ การเก็บตัวอักษรมี 2 รูปแบบหลักๆ ที่ทำงานต่างกัน ดังนี้:

### 🅰️ ตัวแปรแบบ `char` (Single Character)
*   **ทำงานยังไง:** จองพื้นที่หน่วยความจำขนาดเล็กมาก (1 Byte) เพื่อเก็บรหัสตัวอักษรตัวเดียวตามตาราง ASCII
*   **ต้องใส่อะไร:** โค้ดประกาศ `char ch = 'A';` (ใช้ Single Quote เท่านั้น)
*   **ทำเพื่ออะไร:** ใช้ประหยัดพื้นที่เมื่อต้องการเก็บสถานะตัวอักษรตัวเดียว เช่น เพศ (`'M'`/`open`), กลุ่มเลือด (`'O'`)

### 🧵 ตัวแปรแบบ `char[]`
*   **ทำงานยังไง:** เป็นการนำตัวอักษรหลายๆ ตัวมาต่อเรียงกันเป็นสาย (Sequence of Characters) 
    *   ถ้าเป็นแบบภาษา C (`char name[20]`) จะเป็นการจองช่องเก็บตัวอักษรต่อกันเรียงแถว
*   **ทำเพื่ออะไร:** ใช้เก็บข้อความขนาดยาวที่ไม่สามารถจบได้ในตัวอักษรเดียว เช่น การรับพิมพ์ชื่อผู้ใช้งาน หรือข้อความแจ้งเตือนระบบ

## 🚀 ตัวอย่างโค้ดผสมผสาน (Example Code)

```cpp
#include <iostream>
#include <cstdio>
#include <string>

int main() {
    int age;
    float GPA;
    char grade;
    char name[50]; // String ในรูปแบบ Character Array

    // อธิบายการทำงาน: ใช้ printf แสดงคำถาม และ scanf รับข้อมูลตาม Format ของตัวแปรนั้นๆ
    printf("Enter your name: ");
    scanf("%s", name); 
    
    printf("Enter your age: ");
    scanf("%d", &age); // ต้องใส่ & หน้านำทาง int
    
    printf("Enter your GPA: ");
    scanf("%f", &GPA); // ต้องใส่ & หน้านำทาง float

    // สรุปผลลัพธ์กลับไปให้ผู้ใช้
    printf("\n--- Profile ---\n");
    printf("Name: %s\n", name);
    printf("Age: %d years old\n", age);
    printf("GPA: %.2f\n", GPA);

    return 0;
}
